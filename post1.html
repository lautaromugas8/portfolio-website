This was a project I started back in october 2021 when I first learned JavaScript and React. I was the sole creator/developer for the project.

The app got the videogames from an API rawg.io and displayed them on a grid for users to see their name, genres and thumbnail.

Users have the ability to view all their data by clicking on the game, as well as search for the game they want using the search bar or by sorting the games.

Additionally, users can create their own custom game and add it to the list using a form.

I developed this app in less than a month.

Technologies:
  • - JavaScript
  • - React JS
  • - NodeJS + Express
  • - Postgres
  • - Heroku
  • - HTML + CSS
