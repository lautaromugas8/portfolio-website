This was a project I started back in december 2021 as a group project. I developed the Home page, shopping cart, shopping list and more.

The app got the products from our own API and database and displayed them on a grid for users to see their name, price and pictures.

Users have the ability to view all their data by clicking on the product, as well as search for a product, sort them and sell/buy any items.

The payment is done using Mercado Pago.

We developed this app in less than a month.

Technologies:
  • - TypeScript
  • - React JS
  • - Material UI
  • - Firebase
  • - Vercel
  • - NodeJS + Express
  • - MongoDB
  • - Heroku
  • - HTML + CSS
